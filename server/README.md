# @fps/server

Node.js + Socket.io matchmaking service. See `../shared/protocol.md`.

```bash
npm run dev   # tsx watch on port 4000
```

Env vars: `PORT`, `SOLO_TEAM_SIZE`, `TEAM_TEAM_SIZE`, `GAME_SERVER_IP`,
`GAME_SERVER_PORT`, `FPS_SPAWN_GAME_SERVERS`, `FPS_GAME_SERVER_BINARY`,
`FPS_GAME_SERVER_PORT_BASE`.
